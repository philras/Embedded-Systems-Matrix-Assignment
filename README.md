# Embedded_Systems_Matrix_Assignment

A simple text scroller using led matrix and arduino.

Link to youtube demo: <a>https://www.youtube.com/watch?v=r07GszedrXQ&feature=youtu.be</a>

Components:

27 x Cables <br>
1 x MAX7219 <br>
1 x Arduino Uno <br>
3 x 10k resistor <br>
2 x breadboard <br>
